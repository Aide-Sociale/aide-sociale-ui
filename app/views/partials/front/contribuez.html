Contribuez !

Mes-aides est un projet ouvert et participatif. Le code de l'outil est disponbile en Open Source sur GitHub où vous pourrez rejoindre si vous le souhaitez la communauté des développeurs.

Par ailleurs, produire des tests automatisés à partir de cas réels qui fonctionnenent ou ne fonctionnent pas a beaucoup de valeur pour sécuriser et faire évoluer notre outil. En fin de simulation, un bouton est disponible pour sauvegarder de tels cas de tests. N'oubliez pas de documenter vos tests avec les textes réglementaires et les explications qui les justifient.

Bienvenue dans la communauté mes-aides.gouv.fr !
